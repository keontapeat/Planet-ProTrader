SUPER-SIMPLE “ONE-COPY” SETUP  (Native Linux, No Wine Tweaking)
──────────────────────────────────────────────────────────────────────────────
Feeling overwhelmed?  Copy-paste the block below exactly as-is after you SSH
into the VPS.  It does EVERYTHING for Option B automatically: installs
Python 3.10, builds a fresh venv, installs MetaTrader5, then launches a
quick connectivity test.

#────────────────────────  BEGIN COPY BLOCK  ────────────────────────
sudo apt update -y && \
sudo apt install -y python3.10 python3.10-venv tmux wget && \
cd /opt/goldex-bot && \
python3.10 -m venv venv && \
source venv/bin/activate && \
python -m pip install --upgrade pip && \
python -m pip install "MetaTrader5==5.*" pandas python-dotenv && \
echo '✅  All packages installed.' && \
python - <<'PY'
import MetaTrader5 as mt5, os, sys
from dotenv import load_dotenv
load_dotenv('.env')
login    = int(os.getenv('DEMO_LOGIN', '0'))
password = os.getenv('DEMO_PASSWORD', '')
server   = os.getenv('DEMO_SERVER', '')
if not mt5.initialize(login=login, password=password, server=server):
    print("❌  MT5 initialize failed:", mt5.last_error())
    sys.exit(1)
info = mt5.account_info()
print("✅  Connected. Balance:", info.balance)
mt5.shutdown()
PY
#────────────────────────   END COPY BLOCK   ────────────────────────

What it does
1. Updates APT & installs python3.10 and tmux (only once).
2. Creates /opt/goldex-bot/venv with Python 3.10.
3. Installs MetaTrader5 + deps.
4. Runs a 15-line inline Python snippet that:
   • loads credentials from .env  
   • initializes MT5  
   • prints your live balance  
   • shuts MT5 down cleanly.
5. You’ll see either “✅ Connected. Balance: …” or a clear error.

If you get the green check, your environment is ready—just run:

tmux new -s goldex
cd /opt/goldex-bot
source venv/bin/activate
python goldex_bot.py
# Ctrl-b d  (detach)

Done!  No Wine gymnastics required.  Copy, paste, watch it work. 🚀
