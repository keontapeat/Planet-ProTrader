5.  AUTO-START ON VPS REBOOT  (optional but recommended)
────────────────────────────────────────────────────────────────
Create a simple systemd service:

nano /etc/systemd/system/goldex.service

Paste:
--------------------------------------------------------------
[Unit]
Description=GOLDEX AI Demo Bot
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/opt/goldex-bot
ExecStart=/opt/goldex-bot/venv/bin/python /opt/goldex-bot/goldexv9.py
Restart=always
RestartSec=15

[Install]
WantedBy=multi-user.target
--------------------------------------------------------------
Ctrl-O → Enter → Ctrl-X

Enable + start:
systemctl daemon-reload
systemctl enable --now goldex.service

Check log:
journalctl -u goldex.service -f
