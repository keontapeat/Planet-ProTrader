SUPERCHARGE YOUR APP – SUPABASE INTEGRATION + LIVE PLAYBOOK VIEW
================================================================================
This guide ships *everything* you need:

•  iOS / macOS Swift files (with Previews) – drop-in ready  
•  Python additions to goldexv9.py – uploads PNGs to Supabase  
•  Sync logic – app auto-refreshes every 30 s and “learns” (caches) locally  

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
A.  PYTHON BOT  ➜  SUPABASE UPLOAD
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1.  Install lib:
    (venv)$ pip install supabase

2.  Add to **goldexv9.py**  (top, after imports)
    -------------------------------------------------
    from supabase import create_client
    SUPA_URL = os.getenv("SUPA_URL")
    SUPA_KEY = os.getenv("SUPA_KEY")
    supabase = create_client(SUPA_URL, SUPA_KEY)
    -------------------------------------------------

3.  Replace  upload_to_firebase()  with:
    -------------------------------------------------
    def upload_to_supabase(local_path, remote_name):
        with open(local_path, "rb") as f:
            supabase.storage.from_("screenshots").upload(remote_name, f)
        logger.info(f"📤 Uploaded {remote_name} to Supabase")
    -------------------------------------------------

4.  Wherever capture_equity_chart() saved a PNG, call:
        upload_to_supabase(filename, f"screenshots/{filename}")

5.  Export keys once (on VPS):
        echo 'export SUPA_URL="https://xyz.supabase.co"' >> /root/.bashrc
        echo 'export SUPA_KEY="ey...YOUR_SERVICE_ROLE_KEY..."' >> /root/.bashrc
        source /root/.bashrc

6.  scp goldexv9.py to VPS  →  systemctl restart goldex.service

B.  SWIFT PACKAGE  ➜  APP SIDE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    URL: https://github.com/supabase-community/supabase-swift  
    Choose *Up to Next Major*.

2.  Add **SupabaseService.swift**
