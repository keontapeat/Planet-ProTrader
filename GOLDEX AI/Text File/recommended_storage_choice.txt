WHICH FREE STORAGE OPTION FITS *YOUR* APP?
================================================================================
Context
────────
• The PNGs will show account balance / equity and trade timing.  
• You’ll likely view them from your own devices, maybe share a few.  
• They don’t need to be 100 % public, but a simple signed-URL you can open
  in Safari (iOS / macOS) is ideal.  
• You want room to grow (hundreds or thousands of screenshots) without hitting
  an hourly rate-limit.

Verdict → **Supabase Storage**  ✅
────────────────────────────────────────────────────────────────
Why Supabase is the best fit:

1.  Privacy control  
    • Buckets default to private.  
    • You can generate **signed URLs** that expire (or just make the bucket
      “public” if you change your mind).

2.  Generous free quota  
    • 500 MB storage + 2 GB transfer / month  ➜  ~10 000 PNGs at 50 KB each.

3.  No hard rate limits  
    • Uses normal HTTP PUTs; you won’t hit the 50 uploads/hour wall like Imgur.

4.  Easy iOS / macOS integration  
    • Supabase Swift SDK → download / list screenshots inside your future app.

5.  Future proof  
    • Same API works if you later switch to paid Postgres, Auth, Realtime, etc.

Only choose Imgur / GitHub if …
────────────────────────────────────────────────────────────────
• Imgur → You genuinely don’t care about privacy and need *zero* setup.  
• GitHub → You love having version history & public CDN, and the
             screenshots do **not** reveal sensitive balances.

Quick next steps
────────────────
1.  Create free Supabase project → copy  SUPABASE_URL  &  SUPABASE_SERVICE_ROLE_KEY  
2.  Update goldexv9.py with the *supabase* snippet from the previous doc.  
3.  Store the two values as environment variables on the VPS:

        echo 'export SUPA_URL="https://xyz.supabase.co"' >> /root/.bashrc
        echo 'export SUPA_KEY="ey..."' >> /root/.bashrc
        source /root/.bashrc

4.  Re-upload goldexv9.py, restart goldex.service.
5.  Check bucket `screenshots/` in Supabase → images appear after bot runs.

That’s it—Supabase covers privacy, capacity, and easy Swift integration for $0. 🚀
