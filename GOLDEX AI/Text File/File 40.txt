Troubleshooting
───────────────
• If `winetricks -q -f vcrun2022` prints **help again**, your clipboard is
  adding hidden characters.  Type the command manually.  
• If `vcrun2022` isn’t listed, use `vcrun2019` instead:  
  winetricks -q -f vcrun2019  
• Still getting “fetestexcept” after install?  The runtime didn’t register.
  Run:  WINEPREFIX=$HOME/.wine winetricks -q -f vcrun2022 win10
